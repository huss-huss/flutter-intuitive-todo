## 🚀 How to Run

### **Quick Start (Recommended)**
```bash
# 1. Get dependencies
flutter pub get

# 2. Run on web (fastest way to see it)
flutter run -d chrome

# 3. Or use our build script
./build.sh web
```

### **Platform-Specific**
```bash
# Android
./build.sh android

# iOS
./build.sh ios

# Build APK
./build.sh apk
```
